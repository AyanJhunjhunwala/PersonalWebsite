ChordWizard

ChordWizard is a real-time chord recognition system using OpenCV and spectral analysis. Processes audio input through Librosa for MEL spectrogram generation, then uses CNNs to detect and identify guitar chords. Users can create their own progressions and there is also a built in finger position visualizer.

PyTorch Librosa Spectrograms OpenCV

BiteBuddies

BiteBuddies is a full-stack food sharing platform with OCR-enabled receipt scanning using Core ML and Vision frameworks. The Swift-based iOS app implements Core Data and communicates through a RESTful API to a Flask backend. Features include personalized recipe recommendations and a community-based food sharing system to reduce waste.
