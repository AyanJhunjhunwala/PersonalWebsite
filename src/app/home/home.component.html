BiteBuddies

BiteBuddies connects people through food to ease transitions and reduce waste. Users build virtual pantries with OCR-enabled receipt scanning and get recipe suggestions. The iOS app in Swift communicates with a Flask API via RESTful endpoints. The Flask API integrates Spoonacular for food classification and manages user data.

ChordWizard

ChordWizard is a real-time chord recognition system using OpenCV for sound and MEL spectrogram analysis. The system detects and displays the guitar chord. The project is trained on a custom dataset from Kaggle and it is also optimized to recommend chord prgressions and allows you to chose your own.

Reddit Reviewer

This project leverages Beautiful Soup to scrape Reddit threads for real-life user experiences about specific products. The collected data is then processed using an optimized Hugging Face NLP model, allowing for accurate sentiment and topic analysis. Finally, the results are visualized to help users quickly gauge community opinions and make informed decisions.

Monte Carlo Options Pricing

Developed a Monte Carlo simulation in Python to price financial derivatives and assess risk metrics. Generated stochastic price paths for trending tickers, visualized price distributions, and calculated VaR & CVaR to quantify downside risks. Achieved pricing within 2% of expected outcomes
